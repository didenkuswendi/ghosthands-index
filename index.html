📖 Latar Belakang & Tujuan

Pekerjaan kantor modern seringkali membutuhkan ratusan klik, input data berulang, scroll halaman, dan navigasi antar tab setiap harinya. Mulai dari mengisi spreadsheet, mengelola database, approval sistem, hingga memproses laporan – semua itu memakan waktu dan rawan kesalahan manusia.

Ghost Hands lahir untuk mengotomatisasi tugas-tugas membosankan tersebut. Dengan antarmuka yang intuitif, Anda dapat merekam/menyusun skenario step-by-step (klik, ketik, pilih dropdown, scrolling, loop, variabel) dan menjalankannya secara otomatis.

Halaman ini menyediakan berbagai elemen yang bisa langsung diuji coba dengan ekstensi. Gunakan tab Dirty Hands untuk membuat skenario otomatis.

📄 Contoh Skenario JSON (load ke Dirty Hands)

Salin kode JSON di bawah, simpan sebagai file demo-ghost-hands.json, lalu gunakan tombol Load Skenario (JSON) di tab Dirty Hands.

  "name": "Demo Skenario - Eksplorasi Penuh",
  "steps": [
    { "action": "navigate", "param": "https://didenkuswendi.github.io/ghost-hands-test/", "extra": "" },
    { "action": "waitForPageLoad", "param": "", "extra": "" },
    { "action": "click", "param": "#btnClickMe", "extra": "" },
    { "action": "delay", "param": 800, "extra": "" },
    { "action": "type", "param": "#demoInput", "extra": "Halo, ini otomatis!" },
    { "action": "delay", "param": 500, "extra": "" },
    { "action": "selectDropdown", "param": "#demoSelect|Pilihan 2 (Opt2)", "extra": "" },
    { "action": "delay", "param": 500, "extra": "" },
    { "action": "click_by_text", "param": "Edit", "extra": "" },
    { "action": "delay", "param": 500, "extra": "" },
    { "action": "scroll", "param": 300, "extra": "" },
    { "action": "delay", "param": 600, "extra": "" },
    { "action": "click", "param": "#btnLoop-2", "extra": "" },
    { "action": "delay", "param": 400, "extra": "" },

    { "action": "delay", "param": 500, "extra": "" },
