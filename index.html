📖 Cerita di Balik Ghost Hands

Saya adalah seorang praktisi di industri yang setiap hari harus mengisi puluhan data, klik tombol yang sama berulang kali, scroll halaman, dan pindah tab. Rasa capek dan bosan karena pekerjaan yang monoton mendorong saya untuk mencari solusi otomatisasi. Awalnya saya menggunakan berbagai macam native bahasa pemrograman, tapi rekan seperjuangan tidak bisa menggunakannya. Lalu saya coba ekstensi Chrome yang ada, namun tidak fleksibel dan mahal.


Dari situlah saya memutuskan membuat Ghost Hands – sebuah ekstensi otomatisasi yang mudah digunakan, gratis dengan kuota harian, dan premium terjangkau. Saya ingin membantu rekan-rekan pekerja kantor, admin, HR, data entry, developer, dan siapa saja yang lelah dengan klik manual berulang. Dengan Ghost Hands, Anda bisa:


  • ✅ Mengklik elemen berdasarkan teks atau selector CSS
  • ✅ Mengetik teks otomatis (bisa pakai dataset {{line}})
  • ✅ Scroll halaman atau elemen tertentu (termasuk arah kiri/kanan)
  • ✅ Memilih dropdown, hover, upload file, dan banyak lagi
  • ✅ Menggunakan variabel dan loop untuk tugas kompleks

Sebagai programmer, saya banyak membuat system otomasi lintas sistem operasi dengan multi cara, bahkan ekstensi sederhana untuk kebutuhan saya sendiri. Namun teman‑teman non‑coding sering mengeluh: "Ini rumit, saya tidak bisa memakainya!" Karena itu, saya membangun Ghost Hands: alat otomatisasi visual tanpa kode yang dapat dipakai oleh siapa saja – akuntan, admin, HR, marketing, atau siapa pun yang lelah mengulang pekerjaan yang sama setiap hari.


Ghost Hands lahir dari kebutuhan nyata di lapangan. Karena saya langsung bergerak di industri, ekstensi ini akan selalu diperbarui sesuai kasus aktual dan masukan pengguna. Saya ingin pekerjaan Anda lebih ringan, produktivitas melonjak, dan Anda bisa fokus pada hal yang lebih penting.


Ghost Hands gratis selamanya untuk kebutuhan dasar (10 eksekusi step/hari). Jika Anda butuh unlimited dan fitur premium, Anda bisa meningkatkan melalui tab Lisensi di dalam ekstensi. Tidak perlu donasi eksternal – semuanya terintegrasi.

📘 Panduan Lengkap: Cara Install, Izin, Do & Don'ts

🔧 Cara Install Ekstensi

  1. Download file zip dari link di bawah (setelah setujui syarat).
  2. Ekstrak ke folder permanen.
  3. Buka chrome://extensions di Chrome/Edge/Brave.
  4. Aktifkan Developer mode (toggle kanan atas).
  5. Klik Load unpacked, pilih folder hasil ekstraksi.
  6. Ikon Ghost Hands muncul di toolbar. Klik untuk membuka side panel.

⚠️ Catatan: Ekstensi tidak dipublikasikan di Chrome Web Store, jadi akan ada peringatan "Developer mode extensions" setiap kali Chrome dibuka – ini normal.

🌐 Browser yang Didukung

  • ✅ Google Chrome (versi 88+) – paling stabil
  • ✅ Microsoft Edge (Chromium)
  • ✅ Brave, Opera, Vivaldi
  • ❌ Firefox (tidak support side panel API)
  • ❌ Safari

✅ Do & Don'ts

  • ✅ Gunakan untuk otomatisasi kerja, testing, data entry, scraping non‑ilegal.
  • ✅ Backup skenario (simpan JSON) secara rutin.
  • ❌ Jangan gunakan untuk spam, aktivitas melanggar hukum, atau input data sensitif di situs tidak terpercaya.
  • ❌ Jangan bagikan license key Anda.
🎯 Contoh Selector & Format Aksi (Lengkap)

CSS Selector untuk Klik, Ketik, Tunggu, dll

#id                → #submitBtn
.class              → .btn-primary
atribut             → input[name="email"]
tag.class           → div.content
nth-child           → li:nth-child(3) a
nth-of-type         → div:nth-of-type(2)

Scroll (halaman / elemen)

Scroll halaman:       300 (turun), -200 (naik)
Scroll elemen + arah: #scrollArea|down|200
Scroll ke elemen:     #target|intoView

Pilih Dropdown

Berdasarkan teks:   #selectId|Pilihan 2
Berdasarkan value:  #selectId|value:opt2

Ketik teks & gunakan {{line}}

Teks biasa:         #inputId|Halo dunia
Dengan {{line}}:    #inputId|{{line}} (wajib dalam loop auto + dataset angka)

Klik berdasarkan teks (click_by_text)

exact: "Simpan"
wildcard: "*simpan*"   (mengandung)
awalan: "Simpan*"
akhiran: "*Simpan"
indeks: "Simpan | 2" atau "Simpan | last"

Klik semua elemen (click_each)

Parameter: .btn-delete, nilai tambahan (delay antar klik, default 500ms)

Loop

Start Loop:         5 (angka tetap) atau "auto" (sesuai dataset)
Di dalam loop:      gunakan {{line}} untuk mengambil baris dataset

Hover / Mouse Over

Parameter: CSS selector (.dropdown-menu)

Upload File

Parameter: selector input file; nilai tambahan (kosong, buka dialog)

Navigasi & Tab

navigate: URL
click_open_new_tab: selector link
close_tab: (tanpa parameter)

Fitur Lainnya

getVariable: #elementId|text / value / html / href (simpan ke variabel)
delay:         500 (ms)
waitForElement: .modal-content (timeout 30s)
waitForPageLoad: (tanpa parameter)
🧩 Tips & Trik (Maksimalkan Ghost Hands)
  • 🔍 Gunakan tombol Suggest di ekstensi – akan otomatis menampilkan elemen yang bisa diinteraksi (termasuk shadow DOM).
  • 🔄 Loop dengan Dataset: isi dataset textarea dengan baris data (misal 1,2,3), lalu start loop (auto) dan gunakan {{line}} di parameter klik, ketik, atau selector.
  • 📦 Variabel Dinamis: ambil teks/value dari halaman dengan getVariable, lalu pakai placeholder {{namaVariabel}} di step berikutnya (case‑sensitive).
  • ⏱️ Jangan lupa sisipkan delay jika halaman perlu waktu merespon (rekomendasi 500-1000ms).
  • 💾 Simpan skenario ke file JSON setelah selesai menyusun step – bisa di‑load kapan saja.
  • 📋 Dataset bisa di‑load dari file .txt (tombol "Load dari file .txt").
  • 🖱️ Rekam Aktifitas: nyalakan rekaman, lakukan klik/ketik/dropdown secara manual, ekstensi akan mengkonversinya menjadi step.
